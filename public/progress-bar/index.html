Progress Bar

In this blog lets understand how to implement Progress Bar using React fundamentals

Concepts that are used in this are :

  • useState
  • useEffect
  • setInterval

Lets understand the use of these hooks –>

  • useState : Generally useState is mainly used to display the change in UI or store some data whenever an event is triggered by the user.
  • In the case of Progress Bar it is mainly used to change the bar value and its percentage with respect the value your gonna understand better with the code
const [value, setValue] = useState(0);
const [percent, setPercent] = useState(value);
  • useEffect : Generally useEffect is used to perform some side-effects in our website like displaying the fetched API data as soon as the website loads or do some changes in UI when certain part of UI changes . In the case of Progress Bar it is used to change the value of for every 100ms using setInterval() method and it is also used to set the percentage and set range that the Progress bar should have like from 0% to 100% and this percent value needs to be changed everytime the value changes so it has dependency array with it
  useEffect(() => {
    setInterval(() => {
      setValue((prev) => prev + 1);
    }, 100);
  }, []);

The above code actually changes the value by one every 100ms as we know the callback function from useState holds previous value so we are goona use that render the change for every 100ms if we dont use this we cant see the change of value .

 useEffect(() => {
    setPercent(Math.min(100, Math.max(value, 0)));
  }, [value]);

The above code as I mentioned before it sets the percentage to value and creates a range b/w 0% to 100% and render the value of percentage everytime the value changes ie 100ms as mentioned above The logic is important here so learn above it properly

(Math.min(100, Math.max(value, 0))

The above code is gonna make sure the percent value stays within 100% cause first Math.max(value,0) even if this has 250 in that the Math.min(100,250) always goona be 100 so its nice little trick isnt it !!

  • setInterval:Generally it is used to perform some asynchronous function like counter display or something In the case of Progress Bar it is used to display the the change of value for every 100ms If you wanna learn more about it : setInterval

Now lets display the progress bar :

<div className="progress">
      <span style={{ color: percent > 49 ? "white" : "black" }}>
        {percent.toFixed(0)} %
      </span>
      <div
        role="progressbar"
        aria-aria-valuemin={0}
        aria-valuenow={percent.toFixed(0)}
        style={{ width: `${percent}%` }}
      ></div>
</div>

In the above code the span tag actually displays the percentage value and to conditionally render the color of the text we have used ternary operator in our className Later to display the animation of the color we have used div tag which is gonna have green color as its background and the width of the div is gonna conditionally render according to percentage value thats important cause it is the one which is gonna show the animation part of the Progress Bar.
